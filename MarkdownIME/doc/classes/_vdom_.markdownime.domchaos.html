screwUp

  • screwUp(pattern: RegExp, replacement: string | function, keepProxy?: boolean): void
    • Defined in VDom.ts:90

    replace the tags from proxyStorage. this works like a charm when you want to un-render something.

    argument

    {RegExp} pattern to match the proxy content.

    argument

    {boolean} [keepProxy] set to true if you want to keep the proxy placeholder in the text.

    example

    chaos.screwUp(/^<\/?b>$/gi, "") //before: getHTML() == "Hello World" proxyStorage == {1: "", 2: ""} //after: getHTML() == "Hello World**" proxyStorage == {}

    Parameters

    • replacement: string | function
